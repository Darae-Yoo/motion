# 포즈 매칭 게임

TensorFlow.js의 포즈 감지 기능을 이용한 웹 기반 포즈 매칭 게임입니다.

## 🎮 게임 설명

이 게임은 웹캠을 통해 사용자의 포즈를 실시간으로 감지하고, 화면에 표시된 목표 포즈와 얼마나 일치하는지를 측정하는 게임입니다.

## 🎯 게임 규칙

1. **게임 시작**: "게임 시작" 버튼을 클릭하여 웹캠을 활성화합니다.
2. **포즈 매칭**: 왼쪽에 표시된 목표 포즈와 동일한 자세를 취하세요.
3. **일치도 확인**: 하단의 진행 상태바에서 실시간으로 일치도를 확인할 수 있습니다.
4. **레벨 진행**: 일치도가 80% 이상이 되면 자동으로 다음 레벨로 진행됩니다.
5. **게임 완료**: 4개의 레벨을 모두 완료하면 총 소요 시간이 표시됩니다.

## 🚀 실행 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 웹캠 권한을 허용합니다.
3. "게임 시작" 버튼을 클릭합니다.
4. 목표 포즈와 동일한 자세를 취하세요.

## 🛠️ 기술 스택

- **TensorFlow.js**: 포즈 감지 모델
- **BlazePose**: 실시간 포즈 감지
- **HTML5 Canvas**: 포즈 스켈레톤 시각화
- **WebRTC**: 웹캠 접근
- **CSS3**: 모던 UI 디자인

## 🎨 주요 기능

- **실시간 포즈 감지**: 웹캠을 통한 실시간 포즈 추적
- **스켈레톤 시각화**: 감지된 포즈를 실시간으로 화면에 표시
- **일치도 계산**: 목표 포즈와 현재 포즈의 유사도 측정
- **진행 상태 표시**: 실시간 일치도를 진행 상태바로 표시
- **타이머**: 게임 진행 시간 측정
- **레벨 시스템**: 4단계 레벨 진행
- **반응형 디자인**: 모바일과 데스크톱 모두 지원

## ⚠️ 주의사항

- 웹캠이 필요합니다.
- 인터넷 연결이 필요합니다 (TensorFlow.js 모델 다운로드).
- HTTPS 환경에서 실행하는 것을 권장합니다 (웹캠 접근 권한).
- 충분한 조명이 필요합니다 (포즈 감지 정확도 향상).

## 🎯 포즈 감지 정확도 향상 팁

1. **조명**: 밝고 균일한 조명에서 플레이하세요.
2. **거리**: 웹캠에서 적당한 거리(1-2m)를 유지하세요.
3. **배경**: 단순한 배경에서 플레이하세요.
4. **의복**: 몸의 윤곽이 잘 보이는 옷을 입으세요.

## 🔧 문제 해결

### 웹캠이 작동하지 않는 경우
- 브라우저에서 웹캠 권한을 허용했는지 확인하세요.
- 다른 애플리케이션이 웹캠을 사용 중인지 확인하세요.

### 포즈 감지가 되지 않는 경우
- 충분한 조명이 있는지 확인하세요.
- 웹캠 앞에서 충분한 거리를 두고 서세요.
- 단순한 배경에서 플레이하세요.

### 모델 로딩이 실패하는 경우
- 인터넷 연결을 확인하세요.
- 브라우저를 새로고침하세요.
